# 下載 & 安裝擴展程式

## 谷歌瀏覽器

**當前版本: 2.x**

> 訪問谷歌程式商店,請點擊 [[BPassword For Chrome](https://chrome.google.com/webstore/detail/bpassword/bacldcokcfmemiljlckpeokehiloamcj)]安裝.
